We all know how annoying it is to style custom checkboxes, radio buttons, and select boxes.
Not Anymore!
Click [GET THE CSS] and add it to your stylesheet. Then copy the html markup below and add it to your page.

get the css

Default

<span data-checkable="radio"><input type="radio" name="r1"><i></i></span>

<span data-checkable="checkbox"><input type="checkbox" name="r2"><i></i></span>



<span data-checkable="select"><select>
<option>Option #1's title here</option>
<option>Option #2's title here</option>
</select><i></i></span>

[data-checkable]{ background-color: white; border: 2px solid #6690fc; color: #6690fc; }
